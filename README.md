# 宿舍信息管理系统（JavaEE / Spring MVC + MyBatis + JSP） 本项目是一个基于 **Spring MVC + MyBatis + JSP + Tomcat7 + MySQL** 的宿舍信息管理系统，包含管理员和学生两个角色。 --- ## 🌟 项目功能 ### 管理员功能 - 登录、退出 - 管理学生：添加 / 删除 / 修改 / 分配宿舍 - 管理宿舍楼：添加 / 删除 / 修改 - 管理宿舍房间：添加 / 删除 / 修改 - 管理报修：查看所有报修、处理报修 - 修改个人资料 ### 学生功能 - 登录、注册 - 查看个人信息 - 修改密码 - 查看宿舍信息 - 报修申请（提交、查询） - 修改个人资料 --- ## 🏗 技术栈 - **Spring MVC 5.3** - **MyBatis 3.5** - **JSP + JSTL** - **Tomcat 7 / Maven tomcat7-plugin** - **MySQL 8.0** - **JDBC + dbcp2 数据连接池** --- ## 🚀 运行步骤 ### 1. 导入项目 使用 IntelliJ IDEA 打开项目，等待 Maven 下载依赖。 ### 2. 创建数据库 在 MySQL 中执行 database.sql 中的脚本。 ### 3. 修改数据库配置 文件位置：src/main/resources/db.properties jdbc.driver=com.mysql.cj.jdbc.Driver jdbc.url=jdbc:mysql://localhost:3306/dorm?useSSL=false&useUnicode=true&characterEncoding=utf8 jdbc.username=root jdbc.password=123456 ### 4. 运行项目 在 IDEA Terminal 执行： mvn tomcat7:run 访问： http://localhost:8080 --- ## 📂 主要目录说明 - controller/：控制层 - service/ serviceimpl/：业务逻辑层 - mapper/：MyBatis 接口 - entity/：实体类 - WEB-INF/views/：JSP 页面 - resources/：配置文件（Spring + MyBatis） --- ## 👨‍💻 默认账号 ### 管理员： 账号：**admin** 密码：**123456** ### 学生示例： 学号：**20230001** 密码：**123456** --- ## 📝 作者 JavaEE 课程设计 — 宿舍管理系统
